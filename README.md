# 📊 Student Management System

A simple Student Management System using Python and SQLite.

## 🚀 Features
- Add Student
- View Students
- Update Student
- Delete Student

## 🛠️ Tech Stack
- Python
- SQLite

## ▶️ How to Run
1. Install Python
2. Run the file:
   python student.py

## 📌 Project Description
This project manages student data using CRUD operations. It helps understand database handling and backend logic.
